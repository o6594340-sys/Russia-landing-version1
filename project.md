## Технический стек

- Чистый HTML5 + CSS3 (без фреймворков, без JS)
- Один файл: `landing.html`
- Адаптивная вёрстка (брейкпоинты: 900px, 720px, 480px)
- CSS-переменные для цветовой схемы
- SVG-иконки встроены инлайн
