# Plan — Japan Travel Concierge Bot

## Видение

Из узкоспециализированного бота для арабских туристов в Токио — в **универсального консьержа по всей Японии** для туристов со всего мира с реальным бронированием.

---

## Фазы развития

### Фаза 1 — Стабилизация (сейчас)
**Цель:** надёжная работа текущей версии

- [x] Бот работает в Telegram
- [x] Отвечает на арабском/любом языке → английский
- [x] Покрытие: Токио, консьерж-режим
- [ ] Запуск 24/7 на сервере (VPS)
- [ ] Мониторинг — уведомление если бот упал

---

### Фаза 2 — Расширение географии (1–2 месяца)
**Цель:** вся Япония

Добавить города:
- Киото (храмы, рёканы, гейши, чайные церемонии)
- Осака (стрит-фуд, Дотонбори, замок)
- Хиросима / Нара / Никко
- Хоккайдо (лыжи, природа)
- Фудзи / Хаконэ

Технически:
- Расширить системный промпт под всю Японию
- Добавить определение города из запроса пользователя
- Обновить приветствие и команду `/help`

---

### Фаза 3 — Расширение аудитории (2–3 месяца)
**Цель:** не только арабы — все туристы мира

- Убрать арабскую специализацию из промпта
- Добавить мультиязычность: отвечать на языке пользователя (английский, русский, французский, арабский и др.) — или оставить английский как универсальный
- Добавить специальные разделы по запросу:
  - Халяль-еда (для мусульман)
  - Вегетарианское/веганское
  - Бюджетные варианты
  - Люкс-сегмент

---

### Фаза 4 — Реальное бронирование (3–6 месяцев)
**Цель:** бот не просто консультирует, но и помогает забронировать

Интеграции:
| Сервис | Что бронируем | API |
|---|---|---|
| **Booking.com** | Отели | Affiliate API |
| **Viator / GetYourGuide** | Экскурсии, мастер-классы | Partner API |
| **Klook** | Активности, Japan Rail Pass | Affiliate |
| **TableCheck / OpenTable** | Рестораны | API |

Модель:
- Бот даёт ссылку с реферальным кодом → комиссия с бронирования
- Или прямая интеграция с формой оплаты внутри Telegram (Telegram Payments)

---

### Фаза 5 — Монетизация (6–12 месяцев)

**Модель 1 — Affiliate (партнёрские ссылки)**
- Комиссия 3–8% с каждого бронирования отеля/тура
- Пассивный доход, не требует обслуживания

**Модель 2 — B2B (белый лейбл)**
- Продавать бота туристическим агентствам как готовое решение
- Подписка: $50–200/мес за кастомизированного бота под их бренд

**Модель 3 — Premium подписка**
- Базовый бот бесплатный
- Premium ($5–10/мес): персональные маршруты, приоритетные ответы, сохранение избранного
