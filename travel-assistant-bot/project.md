# Hana — Japan Travel Concierge Bot

## Что это

Telegram-бот консьерж для туристов, путешествующих в Японию — прежде всего арабская и русскоязычная аудитория.

Бот **не продаёт** — только консультирует: отели, халяль-рестораны, транспорт, достопримечательности, мастер-классы. Автоматически определяет язык и отвечает на нём.

---

## Целевая аудитория

### Сегмент 1 — Арабские туристы
- **Откуда:** Саудовская Аравия, ОАЭ, Кувейт, Катар, Иордания
- **Возраст:** 25–45 лет, семьи и молодые пары
- **Бюджет поездки:** $3,000–8,000
- **Боль:** языковой барьер, поиск халяль-еды, незнание транспорта
- **Где сидят:** Instagram, TikTok, Snapchat (КСА), YouTube

### Сегмент 2 — Русскоязычные туристы
- **Откуда:** Россия, Казахстан, ОАЭ (диаспора)
- **Возраст:** 28–50 лет, средний класс и выше
- **Бюджет поездки:** $2,500–6,000
- **Боль:** сложная логистика, языковой барьер, планирование маршрута
- **Где сидят:** Telegram-каналы, YouTube, ВК

## Ограничения бота (текущая версия)

- Не бронирует — только консультирует
- Не знает цены в реальном времени
- Не принимает фото и голосовые сообщения
- Не помнит историю между сессиями (только в рамках одного разговора)
- Данные могут устаревать — рекомендует проверять на официальных сайтах

## Запуск

**Windows:** двойной клик на `start_bot.bat`

**Терминал:**
```bash
pip install -r requirements.txt
python bot.py
```

**24/7 (планируется):** Railway.app (бесплатно) или Hetzner €4/мес
