# Hana — Tokyo Travel Concierge Bot

## Что это

Telegram-бот консьерж для индивидуальных туристов из арабских стран, путешествующих в Токио.

Бот **не продаёт** — только консультирует: локации, как добраться, время работы, халяль-рестораны, мастер-классы и всё о Токио.

---

## Технический стек

| Компонент | Решение |
|---|---|
| Платформа | Telegram Bot API |
| AI модель | Llama 3.3 70B (через Groq) |
| Язык | Python 3.13+ |
| Библиотека бота | python-telegram-bot 21.10 |

---

## Ключевые правила бота (persona)

- Имя: **Hana**
- Тема: **только Токио** (отели, рестораны, достопримечательности, транспорт, мастер-классы, халяль)
- Язык ответов: **всегда английский**, даже если вопрос на арабском
- Роль: консьерж, не продавец
- Тон: дружелюбный, лёгкий, как общение с другом
- При непонятном вопросе: задаёт 1–2 уточняющих вопроса или предлагает варианты выбора

## Переменные окружения (.env)

```
TELEGRAM_BOT_TOKEN=...   # от @BotFather в Telegram
GROQ_API_KEY=...         # от console.groq.com (бесплатно)
```

## Команды бота

| Команда | Действие |
|---|---|
| `/start` | Приветствие и знакомство с Hana |
| `/help` | Список тем и команд |
| `/clear` | Сброс истории разговора |

---

## Запуск (локально)

```bash
pip install -r requirements.txt
python bot.py
```

Бот работает пока открыто окно терминала.

---

## Деплой (24/7)

Для постоянной работы нужен VPS-сервер (от $4/мес — DigitalOcean, Hetzner и др.).

```bash
# На сервере
screen -S hana
python bot.py
# Ctrl+A, D — отсоединиться, бот продолжает работать
```

## Изменение поведения бота

Все правила, тон и ограничения бота — в файле `prompts.py`. Там можно:
- Добавить/убрать темы
- Изменить язык ответов
- Добавить другие города
- Поменять имя и личность

## История разработки

- AI провайдер менялся дважды из-за региональных ограничений ЕС:
  - Anthropic API → требует VAT номер для Европы
  - Google Gemini → free tier недоступен в Бельгии
  - **Groq** → работает в ЕС, бесплатно ✓
