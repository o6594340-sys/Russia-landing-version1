## 2. Design Strategy

### The Core Idea

Every competitor in UAE tourism uses the same formula:
**black background + gold accents = "Arabic luxury"**

Insiders Tourism will do the opposite.

> **Light. Airy. Refined.**
> The luxury of space, not the luxury of darkness.

This positions Insiders as the brand for travellers who want authenticity and elegance — not spectacle.

### Reference

The structure and approach are inspired by **oplus.ae** — a premium UAE brand with:
- Clean minimal layout
- Strong typography
- Confident use of white space
- Subtle scroll animations
- No visual clutter

The difference: Insiders keeps its own colour identity instead of copying the dark palette.

## 3. Colour Palette

The palette is built directly from the **Insiders Tourism logo** — the magenta from the word *"tourism"* — then refined into an elegant, airy system.

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| **Primary Accent** | Muted Magenta | `#A8406E` | Buttons, links, icons, highlights |
| **Accent Hover** | Rose | `#C4698E` | Button hover states |
| **Accent Pale** | Blush | `#F2E0E9` | Tags, tints, soft backgrounds |
| **Main Background** | Warm White | `#FAF7F8` | Page background |
| **Alt Background** | Soft Mallow | `#F0E8EE` | Alternating sections |
| **White** | Pure White | `#FFFFFF` | Cards, header |
| **Text — Primary** | Near Black | `#1E1A1C` | Headlines, body |
| **Text — Secondary** | Rose Grey | `#6B5A62` | Subtext, descriptions |
| **Text — Muted** | Dusty Rose | `#9E8A93` | Labels, captions |
| **Border / Line** | Pale Rose | `#E4D6DC` | Dividers, card borders |
| **Hero Accent** | Pale Pink | `#F2B8CF` | Italic text on dark hero |

### Why This Palette Works

- Derived from the existing logo — **brand consistency**
- Warm white backgrounds feel **premium, not clinical**
- Magenta accent is **rare in UAE tourism** — instant differentiation
- Muted tones create **space and breathing room**
- Works equally well for **leisure and corporate (MICE)** clients

## 4. Typography

Two typefaces. One rule: **personality in headlines, clarity in body text.**

### Headings — Cormorant Garamond

```
See the UAE
from the inside
```

| Property | Value |
|----------|-------|
| Family | Cormorant Garamond |
| Weights | Light (300), Regular (400), SemiBold (600) |
| Style | Also used in italic for emotional emphasis |
| Source | Google Fonts — free |
| Character | Editorial, classical, timeless — used by luxury fashion & hospitality brands |

**Why Cormorant:** It is the serif of choice for premium travel, luxury hotels and editorial design. It feels expensive without being pretentious. The italic variant adds warmth and movement to key phrases.

### Body Text — DM Sans

```
Private tours, transfers and experiences
crafted for those who want more than a postcard.
```

| Property | Value |
|----------|-------|
| Family | DM Sans |
| Weights | Light (300), Regular (400), Medium (500) |
| Source | Google Fonts — free |
| Character | Geometric, modern, highly legible at small sizes |

**Why DM Sans:** Clean and neutral — lets the serif headlines breathe. Reads perfectly on screen across all devices.

## 6. Key Design Decisions

### Navigation
- Logo left, menu right — standard premium layout
- "Book Now" button always visible in header
- On scroll: header gains white background + soft shadow
- Mobile: hamburger menu → full-screen overlay

### Hero Section
- Full-viewport height photo
- Dark gradient overlay (bottom-heavy) for text legibility
- Headline uses serif + italic in pale pink
- Two calls to action: primary button + ghost link

### Cards & Components
- Subtle border (`#E4D6DC`) — not shadows by default
- Hover: slight lift (4–5px upward) + soft magenta glow shadow
- All cards: `border-radius: 4px` — minimal, not rounded

### Animations
- **Fade-in + slide-up** on all sections as user scrolls
- **Staggered delays** within grids (0.12s per card)
- **Counter animation** on Stats section (ease-out cubic)
- No aggressive animations — smooth and quiet

### Responsive
- Desktop: 4-column services, 3-column tours, 2-column about
- Tablet (≤1024px): 2-column grids
- Mobile (≤768px): single column, burger menu

## 7. Voice & Copy Direction

The text on the new site will be completely rewritten. The approach:

| Old site | New site |
|----------|----------|
| Lists of features | Short, confident statements |
| "We provide services" | "We live here. That's the difference." |
| Generic tourist language | Insider's perspective |
| Formal | Warm but professional |

**Headline style:** serif italic for emotional words
`See the UAE` *`from the inside`*

**Body text:** short paragraphs, no bullet overload, active voice
