# Бот для визиток

Telegram-бот: фото визитки → автоматически в Google Таблицу.

## Как запустить

1. Открой VSCode в папке проекта
2. Открой терминал (Ctrl + `)
3. Запусти:
```
python bot.py
```
4. При первом запуске откроется браузер — авторизуй Google аккаунт
5. Бот работает пока открыт терминал

**Если запущено два бота и появляется ошибка Conflict:**
```
taskkill /f /im python.exe
python bot.py
```

## Пользователи

| Код | Имя |
|-----|-----|
| OKH | Ольга |
| NSH | Надя |
| DI  | Даша |
| AE  | Настя |

Каждая участница находит бота в Telegram, пишет `/start` и вводит своё имя (Ольга, Надя, Даша или Настя).

## Как пользоваться

1. Отправь фото визитки
2. Бот распознаёт данные и спрашивает "Где познакомились?"
3. Напиши канал (например: ITB 2026, Sales call, WTM London)
4. Бот записывает в таблицу

**Канал запоминается на всю сессию** — следующие визитки сохраняются с тем же каналом автоматически.

Чтобы сменить канал: `/channel WTM London`

- Дубликат (та же компания + тот же человек) → бот предупредит, не добавит
- Новый контакт в известной компании → добавит с пометкой

## Столбцы в таблице

Название компании | Сайт | Страна/Регион | Тип компании | Контактное лицо | Должность | Email | Контактный телефон | Канал знакомства | Добавил

## Файлы проекта

- `bot.py` — основной код бота
- `.env` — токены и настройки (не передавай никому!)
- `oauth_credentials.json` — ключ Google (не передавай никому!)
- `token.pickle` — сохранённая авторизация Google (создаётся автоматически)
- `users.json` — зарегистрированные пользователи (создаётся автоматически)
- `requirements.txt` — список зависимостей

## Если бот не запускается

```
pip install -r requirements.txt
python bot.py
```
