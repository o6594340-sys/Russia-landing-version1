# Процесс: Создание первичного коммерческого предложения
_Дата: 17 марта 2026_

---

## 1. ЧТО ЗАПУСКАЕТ ПРОЦЕСС?

**Триггер:** Входящая заявка от клиента или тендерный запрос

**Типичные данные на входе:**
- Страна / направление
- Даты поездки
- Количество участников
- Тип мероприятия (конференция, инсентив, тимбилдинг)
- Бюджет (иногда указан, иногда нет)
- Особые пожелания (иногда есть, чаще нет)
- Дедлайн подачи (критично: часто 2–3 дня)

**Срочность:** ВЫСОКАЯ. Тендер = 2 дня. Первый ответивший с сильным предложением — выигрывает.

## 3. ЧТО НА ВЫХОДЕ?

1. **Смета** — Excel/Google Sheets с позициями, ценами, наценкой, итогом в нужной валюте
2. **Презентация** (PDF / Gamma ссылка) — визуально сильная, с концепцией и фишками
3. **Сопроводительное письмо** — короткое, с ключевыми аргументами почему выбрать вас
4. **Срок:** всё это — за 2 дня максимум

---

## 4. ГДЕ МОЖНО СЛОМАТЬСЯ?

| Точка риска | Что происходит |
|---|---|
| DMC не отвечает вовремя | Не можешь сделать финальную смету — подаёшь приблизительную или вообще не успеваешь |
| Нет бюджета от клиента | Не знаешь, в какую сторону двигать предложение |
| Клиент меняет запрос после брифа | Вся работа переделывается заново |
| Нет фото под направление | Презентация выглядит слабо |
| "Фишки" оказываются недоступны в датах | Концепция рассыпается на этапе подтверждения |
| Курс валюты меняется | Смета устаревает к моменту отправки |

## 5. ПРЕДЛОЖЕНИЕ ПО АВТОМАТИЗАЦИИ

### Главная идея
Не ждать DMC — иметь **черновик предложения с концепцией и фишками ДО ответа от поставщика**.
Пока DMC считает, у тебя уже готова рамка: концепция, активности, структура презентации.
Когда DMC отвечает — просто подставляешь цифры.

---

### Инструмент 1 — Telegram Bot (быстрый сбор брифа)

**Что делает:**
- Клиент или ты заполняет бриф через бота прямо в Telegram
- Бот задаёт 6–8 вопросов: страна / даты / кол-во / тип / бюджет / пожелания
- На выходе — структурированный бриф автоматически летит тебе в нужный формат

**Эффект:** экономит 20–30 мин на уточнениях при каждом запросе

---

### Инструмент 2 — Claude (генерация концепции и фишек)

**Что делает:**
- Получает бриф → генерирует 3 варианта концепции с "фишками" под страну и тип группы
- Например: _"Япония, 20 человек, инсентив, бюджет средний"_ →
  - Вариант A: Самурайская церемония + мастер-класс по суши + ночная прогулка по Гиону
  - Вариант B: Чайная церемония в рёкане + мастер-класс по каллиграфии + рынок Нисики
  - Вариант C: Кулинарный тур + кастомная якитори-вечеринка + sake-дегустация

**Промпт-шаблон (готов к использованию прямо сейчас):**
```
Страна: [страна]
Тип группы: [корпоратив / инсентив / тимбилдинг]
Кол-во участников: [N]
Бюджет на активности: [примерно]
Тип клиента: [российская компания / международная]

Предложи 3 концепции MICE-программы с уникальными активностями.
Каждая концепция: название + 4–5 активностей + почему это выигрывает тендер.
Активности должны быть: аутентичные, не банальные, реально организуемые,
стоимостью не выше 100–150 EUR на человека за активность.
```

**Эффект:** 5 минут вместо 30–60 мин на генерацию идей

---

### Инструмент 3 — Google Sheets (мастер-смета)

**Что делает:**
- Шаблон с фиксированными блоками: Проживание / Трансфер / Питание / Активности / Прочее
- Автоподсчёт на человека и на группу
- Конвертация валют (формула с актуальным курсом через API или вручную)
- Автоматический итог с твоей наценкой (процент)

**Эффект:** 10 мин вместо 30–45 мин на смету

---

### Инструмент 4 — Gamma / шаблон презентации

**Что делает:**
- Готовый шаблон под каждый тип программы (инсентив / конференция / тимбилдинг)
- Claude генерирует тексты для слайдов за 5 мин по брифу
- Ты только вставляешь фото и цифры

**Промпт для текстов презентации:**
```
Направление: [страна]
Концепция: [название концепции]
Активности: [список]
Кол-во участников: [N]
Тон: вдохновляющий, деловой, без клише

Напиши тексты для слайдов презентации:
1. Обложка (слоган 1 предложение)
2. Концепция (3–4 предложения)
3. Описание каждой активности (2–3 предложения каждая)
4. Почему это предложение (буллеты, 4–5 пунктов)
```

**Эффект:** 20–30 мин вместо 60–120 мин на презентацию

---

### Инструмент 5 — Claude (сопроводительное письмо)

**Что делает:**
- Генерирует письмо за 2 минуты по шаблону
- Адаптирует под клиента и тип тендера

**Эффект:** 5 мин вместо 15–20 мин

## ИТОГ: ДО и ПОСЛЕ автоматизации

| Этап | До | После |
|---|---|---|
| Сбор брифа | 20–30 мин | 5 мин (Telegram Bot) |
| Ожидание DMC | 24–48 часов | Параллельно (не блокирует работу) |
| Концепция и фишки | 30–60 мин | 5–10 мин (Claude) |
| Смета | 30–45 мин | 10–15 мин (Google Sheets) |
| Презентация | 60–120 мин | 20–30 мин (Gamma + Claude) |
| Письмо | 15–20 мин | 5 мин (Claude) |
| **ИТОГО** | **3–5 часов** | **45–75 мин** |
